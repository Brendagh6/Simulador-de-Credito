# 📊 Simulador de Crédito Personal y Top Up (NextFinance)

Este proyecto es una aplicación web interactiva que replica el comportamiento y las reglas de negocio de un simulador de crédito bancario moderno (basado en esquemas de amortización continua tipo *Sistema Francés* con desglose de IVA). 

Permite calcular comisiones por apertura con IVA, obtener el "pago por mil" de referencia y evaluar tres escenarios automáticos de líneas de financiamiento: **100%**, **80%** y **50%**.

---

## 🚀 Características Clave

* **Cálculo de Comisión Automatizado:** Integra el cálculo exacto del IVA (16%) sobre la comisión de apertura seleccionada.
* **Simulación Multi-Línea:** Genera tres alternativas de crédito simultáneas con un solo clic.
* **Tabla de Amortización Dinámica:** Desglose mes a mes de Saldo de Capital, Pago a Capital, Intereses Ordinarios, Pago Fijo Base, IVA y Pago Mensual Total.
* **Diseño Responsivo y Elegante:** Paleta de colores *Deep Teal & Emerald* de aspecto corporativo y premium.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de los formularios y contenedores de datos.
* **CSS3:** Diseño responsivo mediante CSS Grid, Flexbox y variables globales de diseño corporativo.
* **JavaScript (Vanilla JS):** Lógica financiera matemática, manejo del DOM y transiciones de pantalla sin recarga de página.
